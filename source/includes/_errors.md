# 错误代码

<aside class="notice">
Errors consist of two parts: an error code and a message.
Codes are universal,but messages can vary.
</aside>

The GTE API uses the following error codes:

```sell
{
  "code": -10012,
  "error": {
    "patternName": "VALIDATE_PARAM_TYPE_ERROR",
    "data": {
      "paramName": "size"
    }
  }
}
```
